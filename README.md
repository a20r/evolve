# evolve [![Build Status](https://travis-ci.org/chutsu/evolve.png)][1]
evolve is a Gentic Algorithm/Programming implemented in C.

## Installation

    # download evolve
    git clone https://github.com/chutsu/evolve.git
    cd evolve

    # make evolve
    make
